5.1 编辑器

运行下载的安装文件安装编辑器

5.2 注册bitbucket帐号

bitbucket

点击上面的链接注册bitbucket帐号,将帐号发给aaron加入项目组

目前发现国内IP注册无法安装机器人识别控件,请设置代理之后再注册,代理:10.1.2.22:8123

5.3 git客户端

点击下载GIT

点击下载乌龟GIT客户端

点击下载乌龟GIT客户端中文语言包

安装下载的客户端和语言包

在任意空白处单击鼠标右键 -> TortoiseGit -> Settings -> Language 改成中文

在"开始"->"搜索程序和文件"中输入git config –global user.name "你的bitbucket帐号",回车

在"开始"->"搜索程序和文件"中输入git config –global user.email "你常用的邮箱",回车

5.4 使用git下载项目工程

进入希望放置项目文件的目录

登录下面网页中复制项目git地址(右上方,选HTTPS):项目

在任意空白处单击鼠标右键 -> Git 克隆

URL填入项目地址 -> 确定

5.5 使用编辑器打开项目

打开编辑器(Cocos creator)

选择打开其他项目,选择项目路径(如:d:\client)并点击选择文件夹
